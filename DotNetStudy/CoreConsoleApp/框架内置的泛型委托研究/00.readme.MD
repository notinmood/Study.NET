# 说明
框架内置了很多泛型的委托，开发人员可以直接使用，而不必自己再定义了。
- `Func<TResult>`: 表示没有参数并返回一个指定类型结果的方法。
- `Func<T, TResult>`: 表示接受一个参数并返回一个指定类型结果的方法。
- `Func<T1, T2, ..., TResult>`: 泛型Func委托最多可以有16个类型参数，最后一个参数是返回类型，其余的是方法的参数类型。
- `Predicate<T>`: 这是一个特殊的`Func<T, bool>`委托，用于表示接受一个类型参数并返回布尔值的方法，通常用于测试某个对象是否符合某个条件。
- `Converter<TInput, TResult>`: 这是一个`Func<TInput, TResult>`的别名，表示接受一个输入参数并返回一个不同类型结果的转换方法。
- `Comparison<T>`: 这是一个`Func<T, T, int>`的别名，表示接受两个相同类型参数并返回一个整数（通常用于比较）的方法。
